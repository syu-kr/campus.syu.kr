안녕하세요! SYU CAMPUS를 더욱 편리하게 사용할 수 있는 방법을 소개합니다.

**PWA(Progressive Web App)** 기술을 통해 SYU CAMPUS를 마치 모바일 앱처럼 설치하고 사용할 수 있습니다!

---

## PWA란?

PWA는 웹과 앱의 장점을 결합한 기술입니다:

- **앱처럼 빠르다** - 오프라인에서도 사용 가능
- **용량이 작다** - 일반 앱보다 훨씬 가볍다
- **설치가 쉽다** - 앱스토어 거치지 않고 바로 설치
- **자동 반영** - 별도 설치 과정 없이 변경사항 적용

---

## 설치 방법

### iOS (아이폰/아이패드)

1.Safari 브라우저에서 SYU CAMPUS 방문

2.공유 버튼 탭
화면 하단의 공유 아이콘 (사각형 + 화살표)

3."홈 화면에 추가" 선택
앱 이름 확인 후 "추가" 탭

4.완료! 홈 화면에 SYU CAMPUS 아이콘이 생김
다른 앱과 동일하게 사용 가능

**Tips:**

- iOS는 Safari 브라우저에서만 PWA 설치 가능합니다
- Chrome, 파이어폭스에서는 설치 불가능

---

### Android (안드로이드)

### 방법 1: Chrome 브라우저 (권장)

1.Chrome에서 SYU CAMPUS 방문

2.주소창 옆의 설치 버튼 클릭
주소창 우측에 나타나는 설치 아이콘 (또는 메뉴에서 찾기)

3."설치" 확인
팝업에서 "설치" 버튼 탭

4.완료! 홈 화면에 SYU CAMPUS가 설치됨

### 방법 2: Samsung Internet (삼성 브라우저)

1.삼성 인터넷에서 SYU CAMPUS 방문

2.메뉴 (≡) → "앱으로 설치"

3.설치 확인

### 방법 3: 수동 설치

1.메뉴 (≡) → "홈 화면에 추가"
또는 **메뉴 → "더보기" → "홈 화면에 추가"**

2. **앱 이름 확인 후 "추가"**

---

## 설치 후 사용 방법

### 앱 업데이트

- 자동으로 변경사항이 적용됩니다
- 별도의 업데이트 버튼이 필요 없습니다
- 다음에 앱을 열 때 자동으로 새 콘텐츠 로드

### 앱 삭제하기

**iOS:**

- 홈 화면에서 SYU CAMPUS 아이콘 꾸욱 누르기 → "앱 제거"

**Android:**

- 앱 드로어에서 SYU CAMPUS 우클릭 → "제거" (또는 기본 설정에 따라)

## ❓ 자주 묻는 질문

### Q1: PWA를 설치하면 용량이 많이 들어가나요?

**A:** 아니요, 일반 앱보다 훨씬 가볍습니다. 대략 50MB 미만으로 설치됩니다.

### Q3: PWA로 설치 후에도 웹에서 접속할 수 있나요?

**A:** 네, 언제든지 가능합니다. 브라우저에서 접속해도, 설치된 앱에서 열어도 동일합니다.

### Q4: 설치했는데 보이지 않아요

**A:** 다음을 확인해주세요:

- 브라우저 버전 확인
- 기기의 날짜와 시간이 정확한지 확인
- 다시 한번 설치 시도해보기

### Q5: 이미 설치한 PWA를 다시 설치할 수 있나요?

**A:** 네, 앱을 삭제 후 다시 설치할 수 있습니다. 데이터 손실은 없습니다.
