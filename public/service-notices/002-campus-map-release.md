## 🗺️ 신규 서비스: 캠퍼스 지도 출시

**캠퍼스 지도 서비스**가 출시되었습니다!

### 기능 소개

- **40개 이상 건물 정보**: 캠퍼스 내 모든 주요 건물의 위치 확인
- **검색 기능**: 건물명이나 시설명으로 빠르게 검색
- **시설 정보**: 각 층별 시설 안내 (도서관, 카페, 편의점 등)
- **반응형 디자인**: 모바일과 데스크톱 모두 최적화

메인 페이지의 "캠퍼스 지도" 메뉴에서 바로 이용 가능합니다!

---

## 📋 이용약관 최신화

캠퍼스 지도 기능이 추가됨에 따라 이용약관이 다음과 같이 변경되었습니다:

```diff
-   캠퍼스정보: 셔틀버스 시간표, 학식 정보, 도서관 열람실 현황
+   캠퍼스정보: 캠퍼스 지도, 셔틀버스 시간표, 학식 정보, 도서관 열람실 현황

-   통합검색: 모든 공지사항, 일정, 연락처를 통합 검색
+   통합검색: 모든 공지사항, 일정, 연락처, 건물 정보를 통합 검색
```
