Hello from SYU CAMPUS.

SYU CAMPUS now provides an English interface through the `/en` path.

This update is intended to make the service easier to navigate for international students, exchange students, visitors, and anyone who prefers English UI labels.

---

## What changed

The English version includes localized navigation, page titles, buttons, status messages, metadata, and common helper text across the main public pages.

You can now use English UI on areas such as:

- Home
- Academic information
- Campus information
- Notices
- Cafeteria
- Library
- Shuttle and public transit information
- Timetable builder
- Schedule poll
- Graduation requirement checker
- Terms and privacy pages

Users outside Korea may be routed to the English version automatically. You can also change the language anytime from the country and language selector in the footer.

---

## Source content may remain in Korean

Some content is still provided in Korean because it comes directly from official or source data.

Examples include:

- University notice titles and bodies
- Course names
- Cafeteria menu items
- Building and facility names
- Shuttle stops and route names

SYU CAMPUS keeps this source content close to the original data to reduce mistranslation risk and make it easier to verify important information.

For official decisions, deadlines, graduation requirements, class registration, and campus operations, please always check the original university source as well.

---

## This is a first version

The English interface is a first global-facing version of SYU CAMPUS.

Some wording may still be improved over time. If you find confusing labels, untranslated UI text, layout issues, or information that should be explained more clearly, please let us know through the contact form or the GitHub repository.

We will keep improving SYU CAMPUS so it can be useful to more students while still preserving reliable Korean source data.

Thank you.
