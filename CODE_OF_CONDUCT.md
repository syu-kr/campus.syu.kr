# Code of Conduct

## Our Standards

참여자는 서로를 존중하고, 기술적 의견 차이를 사람에 대한 공격으로 만들지 않으며, 건설적인 피드백을 제공합니다. 괴롭힘, 차별, 위협, 개인 정보 공개는 허용되지 않습니다.

## Enforcement

문제가 발생하면 저장소 관리자에게 `singhic_dev@syu.kr`로 비공개로 알려 주세요. 관리자는 상황을 검토해 경고, 참여 제한, 차단 등 필요한 조치를 할 수 있습니다.

이 행동 강령은 저장소의 Issue, Pull Request, Discussion과 프로젝트를 대표하는 모든 공간에 적용됩니다.
